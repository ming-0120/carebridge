# 병원 전화번호와 개원일 정보가 사라진 원인 분석

## 문제 상황
병원 목록 페이지에서 전화번호(`tel`)와 개원일(`estb_date`) 정보가 표시되지 않거나 사라지는 현상

---

## 가능한 원인 분석

### 1. 데이터베이스에 NULL 값이 저장되어 있는 경우
**확인 방법**: 데이터베이스에서 직접 확인
```sql
SELECT hos_id, name, tel, estb_date FROM hospital WHERE tel IS NULL OR estb_date IS NULL;
```

**원인**:
- 병원 모델에서 `tel`과 `estb_date` 필드가 `null=True, blank=True`로 설정되어 있어 NULL 값이 저장될 수 있음
- 템플릿에서 `|default:"-"` 필터를 사용하지만, NULL 값이 제대로 처리되지 않을 수 있음

**해결 방법**:
- 데이터베이스에 NULL 값이 있는 경우, 해당 병원의 데이터를 업데이트
- 또는 템플릿에서 NULL 체크를 더 명확하게 처리

---

### 2. AJAX 페이지네이션 시 데이터가 제대로 전달되지 않는 경우
**확인 방법**: 브라우저 개발자 도구의 네트워크 탭에서 AJAX 응답 확인

**원인**:
- AJAX 페이지네이션 요청 시 서버에서 `hospitals_with_number` 리스트가 제대로 생성되지 않을 수 있음
- 페이지네이션 후 `estb_date_formatted`가 재계산되지 않을 수 있음

**현재 코드 확인**:

**문제점**:
- 코드상으로는 `estb_date_formatted`가 제대로 계산되고 있음
- 하지만 AJAX 요청 시 이 로직이 실행되지 않을 수 있음

**해결 방법**:
- AJAX 요청 시에도 동일한 로직이 실행되는지 확인
- 디버깅 로그 추가하여 데이터 확인

### 3. 템플릿에서 데이터 접근 방식 문제
**현재 템플릿 코드**:
```html:456:504:apps/admin_panel/templates/admin_panel/hospital_list.html
<!-- 전화번호 셀 -->
<td>{{ item.hospital.tel|default:"-" }}</td>

<!-- 개원일 셀 -->
<td>{{ item.estb_date_formatted|default:"-" }}</td>
```

**문제점**:
- `item.hospital.tel`은 직접 접근하는데, `item.estb_date_formatted`는 딕셔너리 키로 접근
- 일관성이 없어서 문제가 발생할 수 있음

**해결 방법**:
- 템플릿에서 일관된 방식으로 데이터 접근
- 또는 뷰에서 모든 데이터를 딕셔너리에 포함

### 5. 페이지네이션 후 쿼리셋이 제대로 로드되지 않는 경우
**확인 방법**: 서버 로그에서 쿼리 확인

**원인**:

- 또는 쿼리셋이 제대로 평가되지 않아 데이터가 누락될 수 있음

**문제점**:
- `tel`과 `estb_date` 필드는 기본적으로 로드되지만, 페이지네이션 후 재평가 시 문제가 발생할 수 있음

## 가장 가능성 높은 원인

### 원인 1: AJAX 페이지네이션 시 `hospital_id` 파라미터가 포함되어 상세 정보만 반환되는 경우
**증상**: 페이지네이션 후 목록은 보이지만 전화번호와 개원일이 사라짐

**원인**:
- 페이지네이션 AJAX 요청 시 URL에 `hospital_id` 파라미터가 포함되어 있으면 서버가 상세 정보만 반환하려고 시도
- 하지만 목록 템플릿이 렌더링되지 않아 데이터가 손실됨

**해결 방법**:
- 페이지네이션 AJAX 요청 시 `hospital_id` 파라미터를 제거하도록 `admin_common.js` 수정
- 다른 페이지(승인대기, 사용자목록 등)와 동일한 패턴 적용

### 원인 2: 템플릿에서 NULL 값 처리 문제
**증상**: 일부 병원만 전화번호나 개원일이 표시되지 않음

**원인**:
- 데이터베이스에 NULL 값이 저장되어 있음
- 템플릿의 `|default:"-"` 필터가 제대로 작동하지 않음

**해결 방법**:
- 템플릿에서 더 명확한 NULL 체크 추가:
```html
<td>{% if item.hospital.tel %}{{ item.hospital.tel }}{% else %}-{% endif %}</td>
<td>{% if item.estb_date_formatted %}{{ item.estb_date_formatted }}{% else %}-{% endif %}</td>
```
